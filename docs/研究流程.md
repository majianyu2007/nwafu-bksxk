# 本地采集产物说明

这份文件记录本仓库的本地采集产物、可提交静态快照和用途，API 细节以 `api.notes.md`、`api.generated.md`、`site-map.generated.md` 为准。

## 产物目录

| 路径 | 用途 |
| --- | --- |
| `artifacts/latest/network.jsonl` | 运行期请求、响应摘要和字段形状 |
| `artifacts/latest/blocked-requests.jsonl` | 本地阻断的写操作请求结构 |
| `artifacts/latest/guard-events.jsonl` | 页面内记录脚本的事件日志 |
| `artifacts/latest/snapshot/page.png` | 页面视觉快照 |
| `artifacts/latest/snapshot/dom.html` | 登录后 DOM 快照 |
| `artifacts/latest/static/` | 本机原始业务 JS 备份，不提交 |
| `static-snapshot/bksxk.nwafu.edu.cn/xsxkapp/static/` | 可提交业务前端脚本快照，供离线提取接口和调用链 |
| `static-snapshot/xkres.nwafu.edu.cn/products/jwfw/xsxkapp/public/` | 可提交公共 JS/CSS/图片快照，供离线查看页面资源 |
| `snapshots/grablessons.sanitized.html` | 可提交离线页面快照，脚本禁用，资源指向 `static-snapshot/` |
| `docs/site-map.generated.json` | 静态站点地图的结构化 JSON |
| `docs/static-assets.manifest.generated.json` | 静态资源快照清单，含大小、SHA-256 和来源 URL |
| `docs/offline-snapshots.manifest.generated.json` | 离线页面快照清单，含 HTML 哈希和本地资源引用 |

## 生成命令

| 命令 | 输出 |
| --- | --- |
| `npm run summarize` | `docs/api.generated.md` |
| `npm run site-map` | `docs/site-map.generated.md` 和 `docs/site-map.generated.json` |
| `npm run static-assets` | `docs/static-assets.manifest.generated.json` |
| `npm run offline-snapshots` | `snapshots/grablessons.sanitized.html` 和 `docs/offline-snapshots.manifest.generated.json` |
| `npm run endpoint-audit` | `docs/endpoint-inventory.audit.generated.md` 和 `docs/endpoint-inventory.audit.generated.json` |
| `npm run go-catalog` | `docs/go-client-catalog.generated.md` 和 `docs/go-client-catalog.generated.json` |
| `npm run request-examples` | 查询、选课、退选、状态轮询的本地参数样例 |
| `npm run sanitize` | 清理本地日志中的敏感字段 |
| `npm run sensitive-check` | 扫描准备提交的文档、脚本和生成物，防止真实敏感值进入仓库 |
| `npm test` | JS 语法、接口覆盖、生成物和静态资源完整性检查 |

## 研究重点

- 用 `site-map.generated.md` 做入口覆盖清单。
- 用 `endpoint-inventory.audit.generated.md` 做端点库存一致性审计：静态资源、站点地图、运行期日志、覆盖矩阵和客户端合同必须能对上。
- 用 `go-client-catalog.generated.md` 做 Go struct 和请求构造器的字段目录。
- 用 `api.notes.md` 汇总字段语义、调用顺序和前端来源位置。
- 用 `request-builders.mjs` 复盘 `querySetting`、`addParam`、`deleteParam` 等字段的 JSON 包装方式。
- 用 `static-snapshot/` 复查前端静态实现，不依赖重新下载 `xkres` 或运行期 `artifacts/`。
- 用 `snapshots/grablessons.sanitized.html` 复查选课页结构、课程卡片容器、筛选控件和模板；这个文件不执行脚本，不会访问正式系统。
- 分享仓库前先运行 `npm run sanitize` 和 `npm run sensitive-check`。前者清理本地采集产物，后者检查准备提交的文件。
- 对添加课程、退选、教材订退等写操作，只研究请求构造和本地 mock，不改变正式系统中的真实状态。
